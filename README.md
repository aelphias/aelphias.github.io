## Push_swap project:
What is the goal?
Sort out a an array of integers, using minimal amount of operations.
I have to write 2 programs: checker and push_swap.
Checker should take a sequance of integers e.g:
$:./checker 3 2 1
After you press Enter it waits for commands separated by push of Enter.
After all commands given press ctrl+D
Checker executes all commands then if an array is sorted it prints "OK"
otherwise "KO".
